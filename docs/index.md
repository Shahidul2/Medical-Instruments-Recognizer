# Medical Instrument Classifier
A simple image classification web application that can classify **10 different types of medical instruments**.

### The supported instruments are:
- Scalpel
- Thermometer
- Forceps
- Bone Saw
- Reflex Hammer
- Ultrasound Probe
- Defibrillator
- Otoscope
- ECG Machine
- Glucometer
